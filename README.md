This is a computational tool to employ dynamic mode decomposition in combination with time-delay embedding (DMD-TDE) to preprocess kymographs.
It is shown that DMD-TDE can remove stationary or diffusive background thereby efficiently siolated actively moving entities from kymographs
generated by live-cell microscopy. The tool is based on this publication:
https://www.biorxiv.org/content/10.64898/2026.03.27.714708v1.full.pdf.
